# 🎨 My Professional Portfolio

## English

A modern, responsive personal portfolio website showcasing projects and skills in web development.

### 📋 Features
- **Responsive Design** - Mobile-friendly interface that works on all devices
- **Dark Mode** - Toggle dark/light theme for comfortable viewing
- **Project Showcase** - Display your projects with filtering by category
- **Contact Section** - Easy way for visitors to get in touch
- **Smooth Navigation** - Seamless page transitions and animations
- **Typing Animation** - Dynamic typewriter effect on the homepage

### 🛠️ Technologies Used
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with flexbox and grid
- **JavaScript (ES6+)** - Interactive features and DOM manipulation
- **Font Awesome** - Professional icons
- **Google Fonts** - Beautiful typography

### 📁 Project Structure
```
portfolio/
├── index.html      # Main HTML file
├── style.css       # CSS styling
├── script.js       # JavaScript functionality
├── README.md       # Project documentation
├── .gitignore      # Git ignore file
└── assets/
    └── images/     # Project images
```

### 🚀 Getting Started

#### Installation
1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio
```

2. Open in your browser:
```bash
# Simply open index.html in your browser
# Or use a local server
python -m http.server 8000
```

### 📱 Sections

1. **Hakkımda (About)** - Introduction and professional goals
2. **Yetenekler (Skills)** - Technical skills and competencies
3. **Projeler (Projects)** - Portfolio of completed projects
4. **İletişim (Contact)** - Contact information and social links

### 🎯 Features Breakdown

| Feature | Description |
|---------|-------------|
| Dark Mode | Switch between light and dark themes |

### 📝 Customization

Edit these files to personalize your portfolio:
- `index.html` - Update your name, bio, and projects
- `style.css` - Modify colors, fonts, and layout
- `script.js` - Add new features or modify existing ones
